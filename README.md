# Twist & Tally

Twist & Tally is a fast, distraction-free scoring app for live contests, balloon jams, and judging situations.

Built for speed, clarity, and reliability during real-world events.

## What it does

- Create multiple contests
- Add and manage entrants
- Tap to tally scores (+1)
- Undo recent tallies (per contest)
- View live leaderboards
- Export results as:
  - PDF (organizer-friendly)
  - CSV (all scores, grouped by contest)
- Backup and restore contests via JSON

## Platform

- iPad-first SwiftUI app
- Designed for live, in-person use
- No accounts, no network required
